Akses & AkunApa format Username saya untuk login?

A: Untuk role Pengguna, gunakan kredensial standar berikut:

  • Username: user@skpd
  • Password awal: user@sihati

Jika instansi Anda menerapkan format khusus, Admin akan memberitahukan secara resmi.

Akses & AkunSaya tidak bisa login meskipun username dan password sudah benar.

A: Pastikan Anda menggunakan URL Web App terbaru yang sudah di-deploy. Jika masih gagal, akun Anda mungkin sedang dinonaktifkan sementara. Silakan hubungi Administrator.

Akses & AkunMengapa link di portal SiHATI membuka tab baru?

A: Ini adalah pengaturan keamanan agar halaman portal (tutorial/FAQ) tetap terbuka, sementara aplikasi utama berjalan di tab terpisah untuk menghindari error dan menjaga sesi.
